# 💳 Order Payments Analysis (Olist E-Commerce Dataset)

## 📌 Overview

This analysis focuses on the `order_payment_dataset` table to evaluate revenue generation and customer payment behavior.

---

## 📊 Dataset Source

* **Dataset:** Brazilian E-Commerce Public Dataset by Olist
* **Platform:** Kaggle
* Link: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

---

## 📁 Table Used

**`order_payment_dataset`**

---

## 📈 Analysis Performed

### 1. Total Revenue

Calculated total income generated from all payments.

### 2. Revenue Excluding Credit Card Payments

Measured income from non-credit card payment methods.

### 3. Number of Credit Card Payments

Counted how many payments were made using credit cards.

### 4. Types of payment methods

Determines different types of payment methods found in the dataset.
